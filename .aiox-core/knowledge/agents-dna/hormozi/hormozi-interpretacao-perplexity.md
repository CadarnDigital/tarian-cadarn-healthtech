# Você é um pesquisador especialista em marketing e vendas. Preciso de uma

especificação técnica modular sobre a metodologia de Alex Hormozi para
criação de ofertas e copywriting. Este documento será usado como base para
construir um agente de IA que: (1) gera copies a partir de briefing e
(2) revisa/otimiza copies existentes — para uma agência de marketing que
atende prestadores de serviços premium (high-ticket), com foco principal
em imobiliárias e secundário em escritórios de advocacia ou clínicas de
estética avançada.

REGRA DE INTEGRIDADE: Cite cada afirmação com a fonte (livro + capítulo,
URL de vídeo/post ou nome do podcast + episódio). Se não encontrar dado
confiável, escreva "não encontrado com fonte confiável". Para conteúdo
inferido de materiais públicos (YouTube, podcasts, posts), sinalize como
"[inferência aplicada — fonte: X]". Mantenha os termos originais em inglês
entre parênteses na primeira menção de cada conceito.

---

MÓDULO 1 — EQUAÇÃO DE VALOR (Value Equation)

Entregue como bloco independente, pronto para ser inserido em um system
prompt de agente de IA. Inclua:

1. A fórmula completa com definição precisa de cada variável
2. Para cada variável: táticas específicas para AUMENTAR (Dream Outcome,
Perceived Likelihood of Achievement) ou DIMINUIR (Time Delay,
Effort \& Sacrifice)
3. Como aplicar a Value Equation para serviços high-ticket — especificamente
para imobiliárias (ex: consultoria de compra/venda de imóvel) e para
escritórios de advocacia ou clínicas de estética
4. Exemplo prático de oferta ANTES e DEPOIS da aplicação da fórmula,
em pelo menos um dos nichos acima

---

MÓDULO 2 — GRAND SLAM OFFERS (framework de construção)

Entregue como bloco independente. Inclua:

1. Definição do que diferencia uma Grand Slam Offer de uma oferta comum
2. Passo a passo para construir uma (identificar problema, listar soluções,
empacotar valor)
3. Stack de ofertas: como empilhar componentes para aumentar valor percebido
4. Bônus (Bonuses): como criar, nomear e posicionar segundo as regras
de Hormozi
5. Garantias e reversão de risco (Risk Reversal): tipos ensinados por ele
e quando usar cada um
6. Naming: framework para nomear ofertas, programas e bônus
7. Exemplos aplicados a imobiliárias e a pelo menos mais um nicho
(advocacia ou estética)

---

MÓDULO 3 — COPYWRITING (padrões e estruturas)

Entregue como bloco independente. Inclua:

1. Estruturas de headline que Hormozi usa recorrentemente (com exemplos
adaptados para serviços high-ticket)
2. Como ele escreve CTAs (chamadas para ação) — padrões e variações
3. Vocabulário e expressões recorrentes (power words, anti-fluff)
4. Regras práticas para eliminar "fluff" de uma copy
5. Framework para escrever bullets de oferta que vendem
6. Pelo menos um exemplo de copy antes/depois da revisão aplicando
esses princípios, no contexto de imobiliária ou advocacia

---

MÓDULO 4 — PRINCÍPIOS INEGOCIÁVEIS

Entregue como bloco independente. Inclua:

1. O que Hormozi considera erro fatal em vendas e marketing
2. Crenças nucleares sobre preço, valor e mercado
3. Visão sobre nicho, avatar e competição
4. Como ele vê a relação entre preço e percepção de qualidade
5. Frases ou afirmações diretas dele (com fonte) que ilustrem cada princípio

---

FORMATO DE SAÍDA PARA CADA MÓDULO:

- Abra com uma definição-síntese em 2–3 linhas (adequada para inserção
direta em system prompt)
- Use headers claros (\#\#, \#\#\#), listas numeradas e tabelas comparativas
onde aplicável
- Feche cada módulo com uma seção "Instruções para o Agente" contendo
regras de comportamento derivadas daquele módulo (ex: "Ao revisar uma
copy, o agente deve verificar se cada variável da Value Equation está
sendo endereçada")

Se não encontrar informação confiável para algum item, indique
explicitamente e não preencha com conteúdo genérico.

Aqui está o relatório completo. Ele foi estruturado como quatro blocos modulares e autossuficientes, prontos para uso direto como system prompts ou documentos de referência do agente.

## O que foi entregue

O documento cobre todos os quatro módulos solicitados com rigor de fonte:

**Módulo 1 — Value Equation:** A fórmula completa com definição precisa das 4 variáveis, táticas específicas para cada uma, tabelas comparativas de aplicação para imobiliárias, advocacia e estética, e o exemplo ANTES/DEPOIS de uma consultoria imobiliária — passando de copy commodity para oferta que aborda todas as variáveis com dados reais.[^1][^2][^3]

**Módulo 2 — Grand Slam Offers:** Passo a passo dos 5 estágios (Dream Outcome → Trim \& Stack), as regras de empilhamento de bônus, os 4 tipos de garantia com quando usar cada um, e a fórmula MAGIC completa aplicada aos três nichos — com container words e exemplos de naming.[^2][^4][^5][^1]

**Módulo 3 — Copywriting:** Os padrões de headline (MAGIC, Paradoxo, Tripla, If-Then), as 5 regras de CTA, a Observable Reality Rule com tabela de substituições, o framework anti-fluff e o exemplo de revisão ANTES/DEPOIS de copy de advocacia trabalhista.[^6][^7][^8]

**Módulo 4 — Princípios Inegociáveis:** Os 5 erros fatais, as 5 crenças nucleares de preço, o exemplo canônico do nicho de gestão do tempo (100x de preço com o mesmo produto), e uma tabela de frases diretas de Hormozi com fonte verificada para cada uma.[^9][^10][^11][^12]

O Apêndice de Integridade lista explicitamente o que é inferência de materiais públicos versus o que foi verificado diretamente — em cumprimento à regra de integridade do briefing.
