# Metricas vs KPIs

## Definicoes

- **Metricas:** numeros absolutos (curtidas, comentarios, visualizacoes, seguidores) — indicam atividade, nao desempenho real. Mostram "o que aconteceu".
- **KPIs (Key Performance Indicators):** indicadores-chave de performance criados a partir de **perguntas estrategicas**. Mostram "se estamos indo na direcao certa".

## Analogia

- **Metricas** = painel do carro (velocidade, combustivel, temperatura)
- **KPIs** = GPS (caminho correto, acelerar, frear, mudar de rota)

## Relacao entre Metricas e KPIs

- Metricas sao a **base** que alimenta os KPIs
- Metricas mostram dados isolados; KPIs combinam dados para gerar insight
- Exemplo: **Taxa de engajamento** = (pessoas engajadas / pessoas alcancadas) x 100
- Quando acompanhada de meta temporal ("aumentar de 1% para 3% em 30 dias"), torna-se instrumento de gestao

## Analise Continua

- Grande falha: olhar dados apenas no final do mes (tarde demais para corrigir)
- Monitorar KPIs **semanalmente** para corrigir rotas
- Se engajamento esta abaixo na metade do periodo: revisar conteudo, impulsionar post performante, repensar formato
- Ferramentas como **MLabs** comparam desempenho com media do mercado (Waze do marketing)

## Principio-chave

Metricas mostram o que aconteceu; KPIs revelam o que precisa ser feito. Perguntar certo, medir com precisao e agir com inteligencia.
