# Método C.A.D.A.R.N.

Framework proprietário da Cadarn Martech para estratégia digital. Aplicável a qualquer cliente.

## As 6 Letras

| Letra | Nome | Fase | O que faz |
|-------|------|------|-----------|
| C | Clareza | Diagnóstico | Painel de Controle (sinais vitais) + Matriz do Gap (Ponto A → Ponto B) |
| A | Arquitetura | Planejamento | Contra-inteligência + Jornadas de decisão + Física do negócio (motores e freios) |
| D | Dados & DNA | Identidade | Doutrina operacional + DNA do especialista + DNA da empresa + Arsenal visual + Engenharia de tráfego |
| A | Ação | Execução | DNA do conteúdo + Linhas editoriais + Sistema de funil + Grade operacional + Árvore de decisão + Protocolo de canais |
| R | Rastreio | Métricas | Plano de batalha + Cronograma + Telemetria + Painel de controle + Cofre de leads + Arsenal de inteligência |
| N | Norma | Governança | Protocolo de entrega + Leis de governança + Termo de homologação |

## Princípios Fundamentais

1. **Dados > Opinião** — Toda decisão ancorada em métricas, não em feeling
2. **Especialista > Generalista** — Cada nicho tem seu método, não existe "receita universal"
3. **Engenharia > Sorte** — Resultados são consequência de processo, não de viralização
4. **Verdade > Agrado** — Entregar o que o cliente precisa saber, não o que quer ouvir
5. **Execução > Planejamento** — "Visão sem execução é apenas alucinação" (Edison)

## Aplicação por Cliente

O método é o mesmo. O que muda:
- DNA do especialista (quem é o cliente)
- Personas (público-alvo do cliente)
- Linhas editoriais (temas do cliente)
- Tom de voz (como o cliente fala)
- Identidade visual (como a marca parece)
- Território (onde o cliente atua)
- Concorrentes (quem disputa o mesmo público)
