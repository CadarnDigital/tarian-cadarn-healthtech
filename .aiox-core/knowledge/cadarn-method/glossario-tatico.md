# Glossário Tático — Cadarn Martech

Terminologia padrão usada em todas as operações. Cada cliente pode ter termos adicionais específicos, mas estes são universais.

## Termos de Operação

| Termo | Significado |
|-------|-----------|
| Plano Tático | Framework de 10 passos para estratégia digital completa |
| Console S.I.C. | Dashboard de métricas (Google Sheets) — cruza mídia com caixa |
| Funil CADARN | Sistema de 5 fases: Atração → Autoridade → Validação → Conversão → Base |
| Árvore de Decisão | 6 nós de validação para qualquer peça de conteúdo |
| Grid Tático | Calendário editorial validado pela Árvore de Decisão |
| Vacina Tática | Conteúdo que neutraliza narrativa de concorrente |
| Cofre de Leads | Classificação de temperatura: Frio → Morno → Quente → Embaixador |
| Capô Aberto | Transparência total sobre defeitos/riscos de um produto/serviço |
| Agente NEXUS | IA estrategista (guarda o Playbook, valida ideias) |
| Agente FLUX | IA auditora (guarda o caixa, analisa dados) |

## Termos de Conteúdo

| Termo | Significado |
|-------|-----------|
| Radar | Conteúdo de alerta (Fase 1 — Atração) |
| Doutrina | Conteúdo educativo/método (Fase 2 — Autoridade) |
| Lastro | Conteúdo de prova/caso real (Fase 3 — Validação) |
| Captura | Conteúdo de oferta/CTA (Fase 4 — Conversão) |
| Base | Conteúdo de relacionamento/retenção (Fase 5 — Base) |
| O Líbero | Peça de conteúdo com "imunidade tática" — pode quebrar a cadência normal |

## Classificação de Leads

| Temperatura | Sintoma | Ação |
|-------------|---------|------|
| Frio | Pergunta preço e some | Automação/Remarketing |
| Morno | Perguntas técnicas, não agenda | Nutrição (Fase 2) |
| Quente | Pede visita, envia documento | Ataque Total (Fase 4) |
| Embaixador | Vem por indicação | Tapete Vermelho (CAC Zero) |
