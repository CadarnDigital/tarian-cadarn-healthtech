# Domínio Healtech
## Guia Completo de Saúde Suplementar

---

> Um guia de leitura para quem quer dominar o mercado de saúde suplementar —  
> não como médico, não como técnico, mas como estrategista.

---

## Como usar este guia

Este guia tem 7 módulos. Cada um é independente, mas a ordem foi pensada para que cada leitura prepare a próxima. Leia assim:

| Módulo | O que você vai entender |
|--------|------------------------|
| **0 — Mapa do Terreno** | O sistema inteiro em uma página. Os personagens, o dinheiro, o por quê. |
| **1 — O Mercado** | O tabuleiro: números, regulação, inflação médica, o Espírito Santo. |
| **2 — A Corretora** | Por dentro da operação: o que faz, como ganha, onde a dor mora. |
| **3 — A Clínica** | Como o atendimento vira dinheiro — e por que frequentemente não vira. |
| **4 — O Comprador** | Quem decide, como pensa, o que o move e o que o trava. |
| **5 — A Jogada Cadarn** | Onde você entra, o que você oferece, o ROI que você entrega. |
| **6 — Vocabulário do Expert** | O dicionário que transforma você em alguém que fala a língua do setor. |

---

## A estrutura de cada módulo

Cada módulo tem três blocos fixos:

**Antes de ler** — Uma pergunta. Carregue ela na cabeça enquanto lê. Ao terminar, você saberá a resposta.

**O texto** — Narrativa fluida. Analogias com o mundo de negócios que você já conhece. Termos técnicos explicados na primeira aparição.

**Frase de ouro** — Uma síntese incompleta. Se você entendeu o módulo, consegue completar com suas próprias palavras. Se não consegue, releia.

## Base deste material

Este guia foi construído sobre o **Corpus IA Saúde Suplementar** — 13 pesquisas conduzidas em 2026 cobrindo o mercado capixaba e nacional de saúde suplementar. Os números são reais, os personagens são reais, os problemas são reais.

O mercado de foco é o **Espírito Santo**, com ênfase na **Grande Vitória** — mas os fundamentos valem para qualquer praça.
